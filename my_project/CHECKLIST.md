# 验收清单

## 核心功能验收
- [ ] 训练稳定性：训练过程无崩溃，日志无异常。
- [ ] 预测生成：`predictions.csv` 文件正常生成。
- [ ] 路径规划：Folium地图正确显示路径，节点弹窗信息完整。
- [ ] 解释报告：`explanation.txt` 和 `explanation.json` 文件生成且内容合理。

## 配置与持久化
- [ ] 参数保存：`config.yaml` 正确保存路由和训练参数。
- [ ] 参数加载：前端控件默认值从 `config.yaml` 回填。

## 性能目标
- [ ] 训练时间：在默认网格下（30×30），10分钟内完成12轮训练。
- [ ] 预测响应：生成预测结果时间 ≤5秒。
- [ ] 路径规划：计算最优路径时间 ≤3秒。
